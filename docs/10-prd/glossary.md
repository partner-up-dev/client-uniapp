# Glossary

- Partner: a participant who joins a PartnerRequest
- PartnerRequest: a request that defines roles and expectations for partners
- PartnerRole: the role a partner applies to or holds in a request
- PartnerApplication: a request to join a PartnerRequest
- PartnerSubApplication: an individual role application inside a PartnerApplication
- Chat: the messaging channel between partners
- Notification: a surfaced event about partner or chat activity
- Explore: the page used to browse partner requests
- Profile: the user account surface for personal details
