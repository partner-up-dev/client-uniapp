# partnerRequestSpecificContentEditor

## 描述

搭子请求特定内容编辑器，根据类型显示不同的编辑器。

## Props

| 属性名 | 类型 | 默认值 | 必填 | 说明 |
|--------|------|--------|------|------|
| type | string | - | 是 | 请求类型 |
| form | object | - | 是 | 表单数据 |

## Events

| 事件名 | 参数 | 说明 |
|--------|------|------|
| change | event | 内容变化事件 |

## 依赖

- rideHailingSpecificContentEditor
- commuteSpecificContentEditor
- travelSpecificContentEditor
