# partnerRequestTypeEditor

## 描述

搭子请求类型编辑器，用于选择请求类型。

## Props

| 属性名 | 类型 | 默认值 | 必填 | 说明 |
|--------|------|--------|------|------|
| showSecondConfirmButton | boolean | false | 否 | 是否显示二次确认按钮 |
| safeAreaInsetBottom | boolean | true | 否 | 底部安全区域 |

## Events

| 事件名 | 参数 | 说明 |
|--------|------|------|
| confirm | selected_type | 确认选择类型事件 |
